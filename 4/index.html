Implementation of Links

Types of Links

Target - _self

Default. Opens the document in the same window/tab as it was clicked

Target - _blank

Opens the document in a new window or tab

Target - _parent

Opens the document in the parent frame

Target - _top

Opens the document in the full body of the window

ABSOLUTE URL

the full URL, including protocol ( http / https ), the optional subdomain (e.g. www ), domain ( example.com ), and path.

RELATIVE URL

A local link (a link to a page within the same website) is specified with a relative URL

mail to link

Use mailto: inside the href attribute to create a link that opens the user's email program
